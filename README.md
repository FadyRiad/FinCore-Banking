# FinCore-Banking
### Spring Boot Banking System - DEPI Capstone Project 

**Project Description:**
A Spring Boot banking application handling accounts, deposits, withdrawals, and transaction history. This project applies OOP, exception handling, automated testing (Selenium), and API testing (Postman).
